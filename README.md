# Heady
Official HeadySystems Inc Repo

## GitHub Copilot Customisation

This project includes custom instructions and configuration to make GitHub Copilot more effective.
- `.github/copilot-instructions.md`: Repository-wide instructions for Copilot.
- `.github/workflows/copilot-setup-steps.yml`: Pre-installation workflow for Copilot environment setup.
- `.github/copilot-mcp-config.json`: Model Context Protocol server definitions.

Copilot will use these files to understand the project's structure and install dependencies.
For further guidance, see the [official Copilot agent documentation](https://docs.github.com/en/copilot/using-github-copilot/github-copilot-chat/using-agents-with-github-copilot-chat).
